Game Jam Projects

Bizzare Adventure — Global Game Jam Malaysia 2026

3D first-person maze horror game created for Global Game Jam Malaysia. Players search for keys, unlock doors, and escape while avoiding a hidden enemy. Mask mechanics reveal or conceal objects, altering navigation and strategy.

Gameplay Highlights
  • Mask-based visibility mechanic
  • Dynamic object reveal/hide system
  • Enemy detection dependent on mask state
  • Key collection and door progression
  • Maze exploration and escape objective
Controls
  • WASD — Movement
  • Mouse — Camera Look
